How to run Scribble

command line
validate scribble global protocol and generate scala API

  • open a terminal (e.g., ctrl-alt-t)
  • from home dir (/home/osboxes.org) to scribble root dir
      cd scribble
  • command line script -- from scribble root dir (/home/osboxes.org/scribble)
      ./scribblec-linmp.sh [Proto name]
    default [Proto name] if non given is assumed to be Proto

concrete example

    $ pwd
    /home/osboxes.org/scribble
    $ ./scribblec-linmp.sh modules/linmp-scala/src/test/scrib/Greeting.scr Proto
